# PaddleOCR With OpenCV for Unity

![PaddleOCRWithOpenCVForUnity](https://github.com/user-attachments/assets/12f5c9e1-1478-4acb-ac8e-66b474d0520d)

## Overview

**PaddleOCR With OpenCV for Unity** integrates **[PaddleOCR](https://github.com/PaddlePaddle/PaddleOCR)** with **[OpenCV for Unity](https://assetstore.unity.com/packages/tools/integration/opencv-for-unity-21088?aid=1011l4ehR)** for end-to-end **OCR** (text detection, classification, and recognition) in **Unity**.

[PaddleOCR With OpenCV for Unity on Unity Asset Store](https://assetstore.unity.com/packages/slug/385314?aid=1011l4ehR)
